%%--------------------------------------------
Write a C function, called {\tt rmsnorm}, that takes as input
\be
\item \(x\), type is (\(M_1, n\), {\tt float})
\item \(w\), type is (\(M_1, n\), {\tt float})
\item \(n\), type is (\(M_0\), {\tt int})
\ee
and returns 
\be
\item \(y\), type is (\(M_1, n\), {\tt float})
\ee
such that 
\be
\item \(\epsilon = 10^{-5}\) 
\item \(s \leftarrow \frac{1}{\sqrt{\epsilon + \frac{\sum {x_i}^2}{n}}}\) 
\item \(y_i \leftarrow w_i \times x_i \times s\) \\ 
  \ee

  \be
\item Produce highly optimized C code.
\item Use AVX2 intriniscs wherever possible.
\item Assume that all pointers, \(x, w, y\), are aligned on 64-byte boundaries
\item Assume that there is no overlap in the memory accessed using \(x, w, y\)
  \ee
